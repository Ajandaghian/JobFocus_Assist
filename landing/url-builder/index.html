How to find geoId or a location name

Use this if you want the LinkedIn URL to target a place without guessing the value.

1 Open LinkedIn Jobs

Go to linkedin.com/jobs and start a search.

in Jobs software engineer
2 Choose a location name

Select a country, region, or city from LinkedIn's location picker. The text you see there can also work in the builder.

United States
3 Copy the geoId

If you want the numeric version, copy the value after geoId= in the LinkedIn URL.

...&geoId=103644278&f_TPR=...
