# Publishing Inputs

It's needed to provide the following information when submitting an extension for publishing. The information is used to validate the extension and to create the store listing.

## Store Listing Information

- Title
- Short description
- Long description
- Category
- Language
- Homepage URL
- Support URL

## Visual Assets

- Store icon, `128x128`
- Screenshots, up to 5
- Screenshot size, `1280x800` or `640x400`
- Small promo tile, `440x280`
- Marquee promo tile, `1400x560`

## Policy and Permissions

- Single purpose description
- Storage permission justification
- Host permission justification
- Remote code answer
- Data usage disclosures
- Privacy policy URL

## Testing and Release

- Test instructions
- Manual verification checklist
- Release notes
- ZIP/package validation
