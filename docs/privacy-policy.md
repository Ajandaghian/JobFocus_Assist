# JobFocus Assist Privacy Policy

Effective date: 2026-06-27

JobFocus Assist is a small non-commercial, community-built Chrome extension for LinkedIn job pages.

## Plain-language summary

- The extension works locally in your browser.
- It reads only the LinkedIn job state that is already visible on the page.
- It does not send LinkedIn content to a server.
- It does not use analytics, advertising, or tracking services.
- It stores only the settings needed to remember your choices in Chrome storage.

## What the extension does

- Viewed cards can be shown with a muted style.
- Applied cards can be shown with a completed green style.
- Other cards are left unchanged.
- The extension does not click buttons, submit forms, or automate applications.

## Stored settings

The only stored values are:

- `viewedEnabled`
- `appliedEnabled`
- `extensionActive`

These settings are used only to restore your popup preferences and extension state on the next browser session.

## What is not stored

- Job IDs
- Job titles
- Company names
- Page URLs
- Search history
- User notes
- Application actions

## User responsibility

- You choose whether to install and use the extension.
- You are responsible for checking LinkedIn pages before using LinkedIn itself.
- You are responsible for following LinkedIn's rules and any other rules that apply to you.

## Relationship

JobFocus Assist is not affiliated with, endorsed by, or supported by LinkedIn.
